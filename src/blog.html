I was even able to use it to help a little girl at a coding convention I volunteered at. It has made resizing a page so much simpler as well. Another thing I’m going to talk about is the different kind of elements. The first is the static positioned element which is the default element in your code. The second is relative positioned element which means the element is position on the page to itself. Meaning if you set a position characteristic it will only apply it to its default position. The third is an absolute element which allows you to position the element anywhere relative to the page such as right, left, top, and bottom. The last is a fixed element which keeps an element fixed on a specific part of the browser window so it wont move when the page is scrolled. Pseudo elements are key phrases that change the design of a specific element on a web page. Examples of this are font, color, and size.
